========================================
  SISTEMA DE BLOQUEIO DE LOGIN DUPLICADO
========================================

🎯 OBJETIVO:
Impedir que clientes compartilhem/revendam suas assinaturas,
permitindo apenas 1 login ativo por usuário.

========================================
📱 CENÁRIO 1: Login Normal (Permitido)
========================================

1. Cliente "joão123" abre o app na TV Box
2. Faz login com usuário e senha
3. App verifica no JSONBin.io: "joão123 está livre?"
4. ✅ SIM! Permite login
5. Salva no JSONBin.io:
   {
     "sessions": {
       "joão123": {
         "deviceId": "abc-123-xyz",
         "deviceName": "TV Box Samsung",
         "loginTime": 1234567890,
         "lastHeartbeat": 1234567890
       }
     }
   }

6. A cada 30 segundos, app envia "heartbeat" (estou vivo)
7. Cliente assiste normalmente ✅

========================================
❌ CENÁRIO 2: Login Duplicado (Bloqueado)
========================================

1. Amigo do cliente tenta logar com "joão123" no celular
2. Faz login com usuário e senha
3. App verifica no JSONBin.io: "joão123 está livre?"
4. ❌ NÃO! Já está logado em "TV Box Samsung"
5. BLOQUEIA login com mensagem:
   "Este usuário já está logado em TV Box Samsung. 
    Desconecte o outro dispositivo primeiro."

6. Amigo não consegue usar! 🚫

========================================
🛡️ PROTEÇÕES DO SISTEMA:
========================================

✅ Bloqueio automático de login duplicado
✅ Timeout de sessão (2 min sem heartbeat = logout)
✅ Admin pode ver TODAS as sessões ativas
✅ Admin pode forçar logout remoto
✅ Detecta se dispositivo fechou o app
✅ Detecta se perdeu internet (via timeout)

========================================

• JSONBin.io - Banco de dados em nuvem
• Grátis até 10.000 requisições/mês
• ~300 usuários simultâneos no plano grátis
• 99.9% uptime
• Backup automático
• API REST simples
